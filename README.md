# 🎮 Tic-Tac-Toe AI (C Language)

A lightweight and efficient Tic-Tac-Toe game written in C. This project demonstrates core programming concepts like arrays, functions, and game logic.

## 🌟 Features
* **Smart AI:** Play against an intelligent computer opponent.
* **Minimax Algorithm:** The AI analyzes all possible moves to ensure it never loses.
* **Clean CLI:** Simple and interactive Command Line Interface.

## 🛠️ How to Compile & Run
1. Open your terminal or Command Prompt.
2. Use a C compiler (like GCC) to compile:
   `gcc "Tic Tac Toe (Vinay singh rajpoot).c" -o tictactoe`
3. Run the executable:
   `./tictactoe`
